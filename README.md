# PocketBase on Zeabur

PocketBase 部署到 Zeabur 平台

## 快速开始

1. 在 Zeabur 控制台导入此 GitHub 仓库
2. Zeabur 会自动识别 `zeabur.yaml` 配置
3. 点击部署按钮

## 部署完成后

- 管理后台: `https://your-app.zeabur.app/_/`
- API 端点: `https://your-app.zeabur.app/api/`

## 首次使用

部署后请在 Zeabur 控制台查看日志，按照提示创建超级管理员账户。
